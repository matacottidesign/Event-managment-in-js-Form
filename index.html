Esempio gestione eventi: Form

Eventi gestiti tramite Javascript

  • First Name
    • Numero minimo di caratteri (Minimo 6)
    • Input di solo contenuto testuale (NO numeri)
    • In caso di input sbagliato: bordi rossi + testo rosso
    • In caso di input corretto: bordi verdi + icona
  • Last Name
    • Numero minimo di caratteri (Minimo 6)
    • Input di solo contenuto testuale (NO numeri)
    • In caso di input sbagliato: bordi rossi + testo rosso
    • In caso di input corretto: bordi verdi + icona
  • Mobile
    • Numero massimo di caratteri (Massimo 10)
    • Input di solo contenuto numerico (NO lettere)
    • In caso di input sbagliato: bordi rossi + testo rosso
    • In caso di input corretto: bordi verdi + icona
  • Email address
    • Input obbligatorio di "@"
    • In caso di input sbagliato: bordi rossi + testo rosso
    • In caso di input corretto: bordi verdi + icona
  • Password
    • Numero minimo di caratteri (Minimo 6)
    • In caso di input sbagliato: bordi rossi + testo rosso
    • In caso di input corretto: bordi verdi + icona

Eventi gestiti tramite HTML

  • Last Name
    • Controllo compilazione obbligatoria (required)
  • Mobile
    • Controllo compilazione obbligatoria (required)
    • Controllo compilazione corretta del pattern (10 numeri: XXX XXX XXXX)
    • Controllo compilazione corretta dell'input (No lettere + caratteri speciali)
  • Email address
    • Controllo compilazione obbligatoria (required)
    • Controllo compilazione corretta dell'input (@ obbligatoria + testo dopo @)
  • Password
    • Controllo compilazione obbligatoria (required)
  • Checkbox
    • Controllo compilazione obbligatoria (required)
